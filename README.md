# SnakeGame
simple snake game for school, only russian translation
простая змейка для школы, только русский перевод

известные проблемы:
1. иногда внутри змеи может появиться яблоко, в этом случае змея пройдёт сквозь себя, съедая его
2. наблюдается низкая производительность из-за использования WPF анимации и System.Timers.Timer
3. код разбросан хаотично и не везде отформатирован, читать может быть неудобно, сделайте скидку школнику
